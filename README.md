Para instalar las dependencias:

```shell
$ yarn install
```

Para arrancar el servidor de desarrollo:

```shell
$ npm run serve
```

Para construir el proyecto a desplegar:

```shell
$ npm run build
```

Con cada _commit_ se despliega en [iteisa.netlify.com](iteisa.netlify.com).
